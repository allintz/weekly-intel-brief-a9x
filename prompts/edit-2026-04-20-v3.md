## The change

Every table that displays prediction markets needs a new column: **Market created** — the date the market was first opened for trading. This contextualizes whether a big 1-week swing reflects genuine sentiment shift vs. a brand-new market with no prior price history.

### Tables that need the column (find all of them):

5. GOP Nominee 2028 (Section 2)
6. Community Forecasts (Section 3 — Metaculus / Manifold rows)
7. AI-related prediction markets (Section 4, the consolidated table)
8. Any other prediction market table in the document

### Column placement

Add as the second-to-last column, before "Source" (or wherever the rightmost column is). Column header: `Created`. Format dates as `Mon YYYY` (e.g. `Feb 2026`, `Mar 2025`). If the market is older than 1 year, `YYYY` alone is fine (`2024`).

### Data source — Polymarket

For Polymarket markets, fetch via:
```bash
curl -sf "https://gamma-api.polymarket.com/markets?slug=SLUG" | python3 -c "import json,sys;d=json.load(sys.stdin); m=d[0] if isinstance(d,list) else d; print(m.get('createdAt','') or m.get('startDate','')[:7])"
```

`createdAt` is ISO8601 (e.g. `2026-02-14T12:34:56Z`). Convert to `Feb 2026`.

For multi-market events (like `balance-of-power-2026-midterms` or `democratic-presidential-nominee-2028`), use `/events?slug=` and pull `createdAt` from the event — individual nominee markets in the same event share that created date.

### Data source — Metaculus

Metaculus API: `https://www.metaculus.com/api2/questions/QUESTION_ID/` returns `created_time` or `publish_time`. Convert to `Mon YYYY`.

### Data source — Kalshi

Kalshi markets have `create_time` field. If a specific market isn't tagged with source API here, best-effort: check the Kalshi URL for the market's "opened" date on the UI page.

### Data source — Manifold

`https://api.manifold.markets/v0/slug/SLUG` returns `createdTime` (ms since epoch).

### If data unavailable

If you cannot fetch the createdAt from the API, leave the cell blank (empty) rather than inventing a date. Don't write "unknown" or "N/A" — just empty.

- Normal column styling, same font as other numeric cells (`class="p"` or whatever matches)

- For markets created more than 1 year ago, just show the year
